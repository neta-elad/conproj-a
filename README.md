# How to get started

1. Create a virtualenv
    > python3 -m venv .env

1. Activate it
    > source .env/bin/activate

1. Install Pyconlang
    > pip install git+https://github.com/neta-elad/pyconlang

1. Compile the project (this may take some time)
    > pyconlang book compile

1. View book
    > open .pyconlang/output.html
