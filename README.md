# How to get started

- Create a virtualenv
    > python3 -m venv .env

- Activate it
    > source .env/bin/activate

- Install Pyconlang
    > pip install git+https://github.com/neta-elad/pyconlang

- Compile the project (this may take some time)
    > pyconlang book compile

- View book
    > open .pyconlang/output.html
